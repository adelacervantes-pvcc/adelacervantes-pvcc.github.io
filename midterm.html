Operating Systems Terminology
TERM DEFINITION
Operating System An operating system is software that serves as a bridge between computer hardware and users. It manages and coordinates various aspects of a computer system, including processor, memory, files, devices, security, and user interface. Key functions include task scheduling, memory allocation, file management, device control, security enforcement, and providing a user interface. Popular examples include Windows, macOS, Linux, Android, and iOS.
Job Queue A system for organizing and managing a sequence of tasks or jobs that need to be processed by a computer. Jobs are submitted to the queue, processed in the order they are received, and typically involve batch processing without direct user interaction. The operating system allocates resources to each job as it moves through the queue, and administrators can monitor and control the execution of jobs. Job queues are common in environments where efficient resource utilization is important.
FIFO Stands for "First In, First Out," is a principle used in organizing data structures like queues. In a FIFO queue, the first element added is the first one to be removed. This principle is akin to waiting in line, where the person who arrives first is served first. It is a fundamental concept in computer science and is applied in various algorithms and data structures, ensuring that items are processed in the order they arrive.
Interactive Processing Interactive processing refers to a computing mode where users interact with a computer system in real-time, providing input and receiving immediate feedback.
Time-Sharing Time-sharing is a method where multiple users share a computer system simultaneously. The operating system rapidly switches between users, giving the illusion of each user having continuous access.
Multiprogramming Multiprogramming is a concept where multiple programs are loaded into the computer's memory simultaneously, allowing the CPU to switch between them for execution.
Application Software Application software refers to software designed for specific tasks or applications, such as word processors, web browsers, and graphic design tools.
Utility Software Utility software provides additional functionalities to the operating system or aids in managing system resources. Examples include antivirus programs, disk cleaners, and backup utilities.
Graphical User Interface (GUI) A GUI is a visual interface that allows users to interact with a computer using graphical elements such as icons, windows, and buttons, making it more user-friendly than a text-based interface.
Shell A shell is a command-line interface that allows users to interact with the operating system by entering commands. It interprets user inputs and executes corresponding operations.
Kernel The kernel is the core part of an operating system responsible for managing system resources, providing a bridge between application software and hardware.
Device Driver A device driver is software that facilitates communication between the operating system and hardware devices, allowing them to function properly.
File Manager A file manager is software that provides a user interface for managing files and directories on a computer system.
Memory Manager The memory manager is a component of the operating system responsible for managing the computer's memory, including allocation and deallocation of memory space.
Virtual Memory Virtual memory is a memory management technique that uses both RAM and secondary storage (like a hard drive) to create an illusion of a larger memory space.
Bootstrapping Bootstrapping, or booting, is the process of loading the operating system into the computer's memory when the system is powered on or restarted.
ROM (Read-Only Memory) ROM is a type of non-volatile memory that retains its contents even when the power is turned off. It typically stores firmware or essential system instructions.
Deadlock Deadlock is a situation in computing where two or more processes are unable to proceed because each is waiting for the other to release a resource.
Spooling Spooling is a process where data is temporarily stored in a buffer or queue to smooth out differences in data processing speeds between devices.
Super User (or Administrator) The super user, often referred to as an administrator, is a user with elevated privileges, allowing them to perform tasks that regular users cannot, such as system configuration and maintenance.
Multi-Factor Authentication Multi-Factor Authentication (MFA) is a security measure that requires users to provide multiple forms of identification (factors) to access a system, enhancing security compared to a single authentication method.
