Programming Languages
TERM DEFINITION
Machine Learning Uses data and algorithms to imitate human learning, improving accuracy over time.
Debugging Finding and fixing errors or bugs in software source code.
Assembler Converts basic instructions into bits for the computer's processor to perform operations.
Assembly Language Low-level programming language communicating directly with the computer's software.
Translator Converts a computer program from one language to another.
Compiler Translates a programming language's source code into machine code, bytecode, or another language.
Interpreter Directly executes high-level language instructions without converting to machine code.
Cross-Platform Software Software implemented and run on multiple platforms.
Natural Language Languages spoken naturally, like English, Spanish, and French.
Formal Language Languages designed by people for specific applications; programming languages are formal languages.
Machine Independence Software not dependent on a particular machine's properties.
High-Level Programming Language Allows humans to write computer programs without specific knowledge of hardware.
Object-Oriented Programming (OOP) Organizes software design around data or objects, rather than functions.
Scripting Language Instructions written for a run-time environment to add functions to applications and integrate systems.
Data Types: Integer Positive or negative whole numbers (e.g., 4 or -20).
Data Types: Float Real numbers like 3.14 or -2.35.
Data Types: String Text written in single or double quotes.
Primitive Data Type Basic data types from which all other data types are constructed.
Data Structure Methods of storing and organizing data for more efficient operations in a computer system.
Python Collection Data Structures: List Stores multiple items in a single variable.
Python Collection Data Structures: Tuple Stores multiple ordered and unchangeable items in a single variable.
Python Collection Data Structures: Set Stores multiple unordered, unchangeable, and unindexed items in a single variable.
Python Collection Data Structures: Dictionary Stores data values in key-value pairs; ordered, changeable, and does not allow duplicates.
Function A sequence of program instructions packaged as a unit to perform a specific task.
Local Variable A variable accessible only within a specific part of the program.
Global Variable A variable with global scope, visible throughout the program unless shadowed.
Source Program A text file containing instructions written in a high-level language.
Object Program A computer program translated from equivalent source code into machine language by a compiler or assembler.
